# IS490 Blog

A simple blog to document course updates and progress for IS490.

## Structure

- `index.html` - Homepage
- `updates.html` - Course updates page
- `about.html` - About the course
- `style.css` - Styling for all pages

## How to Use

1. Open `index.html` in a web browser to view the blog
2. Add new updates to the `updates.html` page as the course progresses
3. Customize the content as needed

## Deployment

This blog can be deployed to GitHub Pages for easy sharing with instructors.
